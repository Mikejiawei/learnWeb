# 属性操作

## 非表单元素属性

href、title、 id、src、className

```javascript
var link = document.getElementById('link');
var img = document.getElementById('img');
console.log(link.id);
console.log(link.href);
console.log(link.title);
console.log(img.src);
console.log(img.alt);
```
